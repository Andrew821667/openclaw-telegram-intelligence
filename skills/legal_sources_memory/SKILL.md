# Legal Sources Memory

## Когда использовать
Используй этот skill, когда пользователь просит:

- обзор судебной практики
- что нового в законодательстве
- что обсуждают юристы
- найти тему, кейс, позицию, изменение, практику в правовых источниках

## Важные правила
- Не предлагай пользователю "показать контуры" и не объясняй внутреннюю механику memory.
- Не говори про shell, exec, пути, session files, scopes, API и внутренние инструменты.
- Возвращай только полезный итоговый результат.

## Используемая команда
Используй локальную команду:

`/Users/openclaw/telegram-memory/bin/telegram-memory-tool`

через exec tool.

## Контуры

### Судебная практика
Контур: `legal_cases`

Используй для запросов:
- обзор судебной практики
- судебная практика за период
- поиск по кейсам, спорам, позициям судов

### Новинки законодательства
Контур: `legal_law_updates`

Используй для запросов:
- что нового в законодательстве
- изменения правил
- правовые новинки
- regulatory updates

### Комментарии и обсуждения
Контур: `legal_commentary`

Используй для запросов:
- что обсуждают юристы
- комментарии по теме
- практические мнения и разборы

## Команды

### Brief
`/Users/openclaw/telegram-memory/bin/telegram-memory-tool brief --scope <scope> --days <days> --limit 7`

### Summary
`/Users/openclaw/telegram-memory/bin/telegram-memory-tool summary --scope <scope> --days <days> --limit 10`

### Search
`/Users/openclaw/telegram-memory/bin/telegram-memory-tool search --scope <scope> --query "<query>" --days <days> --limit 10`

## Сопоставление пользовательских запросов

### Судебная практика
- "дай обзор судебной практики за 14 дней"
- "что нового в судебной практике"
- "найди в судебной практике всё про субсидиарную ответственность"

=> используй `legal_cases`

### Законодательство
- "что нового в законодательстве"

=> используй `legal_law_updates`

### Комментарии юристов

- "покажи комментарии по доверенности"
- "какие мнения есть по вопросу"

=> используй `legal_commentary`
