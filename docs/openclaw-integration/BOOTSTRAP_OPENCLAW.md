# BOOTSTRAP OPENCLAW

## Repo role
Этот репозиторий хранит telegram intelligence / scopes / search / brief / summary / integration docs.

## Does NOT store
- bot tokens
- ~/.openclaw/openclaw.json
- runtime sessions
- logs
- live message databases

## After clone
1. Create venv
2. Install dependencies
3. Configure local secrets manually
4. Create / update scopes
5. Sync required scopes
6. Copy skill source files into OpenClaw agent workspaces or symlink them
7. Verify assistant_bot and legal_bot behavior
