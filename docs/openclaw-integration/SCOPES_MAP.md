# SCOPES MAP

## legal_ai
Контур для legaltech / AI / рынка / новостей.
Используется для обзоров по Legal AI и legaltech.

## legal_sources
Общий внешний правовой контур.
Используется как агрегирующий слой внешних правовых источников.

## legal_cases
Контур судебной практики.
Используется legal_bot для обзоров и поиска по судебной практике.

## legal_commentary
Контур комментариев, обсуждений и прикладных мнений юристов.
Используется legal_bot для обзоров и поиска по мнениям и обсуждениям.

## legal_law_updates
Контур изменений законодательства и правовых новинок.
Используется legal_bot для обзоров по изменениям права.

## legal_memory
Контур под рабочую правовую память.
Создан как каркас, пока почти не наполнен.
